How to run it
At the time of writing, a few of us students tried to create a .jar file of the program to be able to run
it from the command line without having to manually import all the referenced libraries. However,
even though Eline did manage earlier together with her coach, we were not able to manage this by
ourselves. Unfortunately, therefore, my program has to be run from Eclipse or some other IDE. To
run it, follow these steps;
• Start the class Server
• Start the class Client for a client, answer some I/O questions
• Start another Client to run a game
